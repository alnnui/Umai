════════════════════════════════════════════════════════════════
                    💳 НАСТРОЙКА ПЛАТЕЖЕЙ
════════════════════════════════════════════════════════════════

Сейчас используется ТЕСТОВЫЙ режим.

Для РЕАЛЬНЫХ платежей:

1. Зарегистрируйтесь на cloudpayments.kz
2. Пройдите верификацию
3. Получите Public ID
4. Откройте файл .env
5. Замените:
   VITE_CLOUDPAYMENTS_PUBLIC_ID=ваш_реальный_id
6. Запустите: npm run build

📖 Подробнее: CLOUDPAYMENTS_SETUP.md

════════════════════════════════════════════════════════════════
                    ❓ ПОМОЩЬ
════════════════════════════════════════════════════════════════

Не вижу кнопку "Admin":
→ Проверьте SQL-запрос выполнен
→ Выйдите и войдите снова
→ Очистите кэш (Ctrl+Shift+Delete)

События не появляются:
→ Проверьте статус "Upcoming"
→ Дата должна быть в будущем
→ Заполнены Title, Description, Speaker (EN)

Платежи не работают:
→ См. CLOUDPAYMENTS_SETUP.md
→ Используйте тестовые карты
